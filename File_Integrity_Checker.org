* Project Idea
A program to record and verify if a file was modified.
** Features
*** Modes:
- Record - tracks the:
  1. Filename
  2. File extension
  3. File contents
  4. Time created
  5. Time last modified
  6. File hash
- Verify - compares:
  1. Current file to existing (if present)
  2. Displays any differences
